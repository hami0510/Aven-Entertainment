# Aiven Entertainment 관리 시스템

엔터테인먼트 기획사 운영 전반(연습생 육성, 경영관리, 콘텐츠·홍보, 보고)을 위한
Streamlit 기반 통합 관리 대시보드입니다.

## 구성

| 페이지 | 내용 |
|---|---|
| 통합 대시보드 (app.py) | 전체 현황 KPI, 다가오는 일정/콘텐츠 요약 |
| 연습생 관리 | 연습생 등록·목록, 보컬/댄스/랩/태도 평가 및 성장 추이 |
| 트레이닝 스케줄 | 트레이닝 세션 등록 및 목록 조회 |
| 경영관리 | 예산/재무, 계약 현황, 전사 일정 |
| 콘텐츠·홍보 | SNS/채널별 콘텐츠 캘린더 |
| 보고서 | 회의록 작성 및 조회 (핵심 결정사항 상단 배치) |

데이터는 `data/aiven.db` (SQLite)에 저장되며, 앱 최초 실행 시 자동으로 테이블이 생성됩니다.

## 로컬 실행 방법

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 로 접속합니다.

## 배포 (Streamlit Community Cloud 기준)

1. 이 저장소를 GitHub에 push
2. https://share.streamlit.io 접속 → New app
3. 저장소 선택, Main file path에 `app.py` 지정
4. Deploy

> 무료 플랜의 경우 파일 시스템이 재배포/재시작 시 초기화될 수 있어
> `data/aiven.db`에 쌓인 데이터가 유실될 수 있습니다. 운영 단계에서는
> 별도의 영구 DB(Postgres 등)로 전환을 권장합니다.

## 법률·세무 관련 유의사항

- 본 시스템의 계약서/재무 내역은 **관리·기록용 도구**입니다.
  법적 효력이 필요한 최종 계약서는 반드시 변호사 등 법률 전문가의 검토를 거쳐야 합니다.
- 재무·세무 관련 기록은 참고용이며, 실제 신고·세무처리는 세무사 확인을 권장합니다.
- 미성년 연습생이 포함될 수 있는 프로젝트 특성상, 트레이닝 시간·콘텐츠 노출 등에서
  대중문화예술산업발전법 등 관련 법규 및 청소년 보호 기준을 항상 확인해주세요.
